### to start the application
```npm install```  
```npm run dev```
- create in .env file the real values of:
```
DB_USERNAME=
DB_PASSWORD=
```
- Prisma commands  
``` npx primsa```  
``` npx primsa db push```  
``` npx primsa generate```  
or ``` npx primsa migrate dev --name init```  
``` npx prisma studio```
